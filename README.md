# موقع أخبار كأس العالم 2026 🏆

موقع ويب تفاعلي بسيط وجميل لعرض تفاصيل وأخبار بطولة كأس العالم القادمة FIFA 2026.

## 💡 فكرة المشروع
موقع إخباري يهدف إلى تقديم معلومات سريعة وحية حول بطولة كأس العالم 2026 المقامة في ثلاث دول (الولايات المتحدة، كندا، المكسيك)، ويحتوي على عداد تنازلي تفاعلي لعرض الوقت المتبقي بالثواني والدقائق والساعات والأيام لضربة البداية للبطولة، مع تصميم متجاوب (Responsive Design) وتجربة مستخدم مميزة.

## 🛠️ الأدوات المستخدمة
- **HTML5**: لهيكلة وعرض محتويات الصفحة بشكل منظم ومفهوم لمحركات البحث (SEO).
- **CSS3**: لتنسيق الصفحة وإعطائها طابعاً رياضياً وعصرياً باستخدام تأثيرات بصرية وانتقالات سلسة (Transitions).
- **JavaScript (ES6)**: لبرمجة العداد التنازلي التفاعلي المباشر للبطولة.

## 📁 هيكلية المشروع
```text
Project/
├── index.html            # الصفحة الرئيسية للموقع

├── js/

├── images/
│   └── world cup 2026.png # صور وعناصر الوسائط المستخدمة في الموقع

```
